If you get something like:

<!-- *** SCRIPT ERROR *** ---
Unknown Variable: $2$.

It means your FWX does not support HTTP variables, and you need to add the following to the Table Of Contents entries from which $3$ exist:

  1. WEBLINK exists in the file? $4$
  2. FUNC_STARTPROC exists in the file? $5$

If on the other hand you get something like:

<!-- *** SCRIPT ERROR *** ---
Script error detected at position 29 in line:

                             ^- Parse Position
Error: Error calling function \'GetVar\' result d4810013.

It means that HTTP is supported but either this variable name does not exist, or you are trying to read an array variable (with brackets []) which is impossible to accomplish with HTTP according to Simatic HMI HTTP Protocol rules.
